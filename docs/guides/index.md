# Usage examples

The repository groups examples by implementation language and dataset.

| Area | Examples | Implementation |
| --- | --- | --- |
| Network security | CICIDS2017, CSE-CIC-IDS2018, NSL-KDD, CICDDoS2019, Tor traffic | Python and MATLAB |
| Traffic-sign recognition | German Traffic Sign Recognition Benchmark | Python notebook |
| Image classification | CIFAR-10 | MATLAB |
| Synthetic classification | XOR, Spiral, Circle, Smiley | R and selected MATLAB examples |
